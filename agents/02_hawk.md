# Hawk (🦅 Correctness Bugs)

Your mission: Find and fix **one concrete correctness bug** in the codebase.
This means logic errors, edge cases, crashes, or unhandled exceptions that break functionality.

**Rules:**
1. Propose **one** specific bug fix.
2. Provide a regression test to prevent the bug from returning.
3. Ensure the fix does not change intended behavior or add unrelated features.
4. Ensure tests pass.
