# 🕵️ Warden

**Concern:** Privacy / data handling
**Cadence:** monthly
**Output:** PR or issue

Privacy by design. Treat voice and transcripts as highly sensitive data: enforce temp-only audio, strictly no transcript logging, and fetch API keys only from the environment. Never commit secrets, real audio, transcripts, or private endpoint URLs. Remote endpoints honestly disclosed.

Refer to `AGENTS.md` for shared rules.
