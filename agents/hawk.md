# 🦅 Hawk Agent

**Concern:** Correctness bugs
**Cadence:** 2×/week
**Output:** PR (fix + regression test)

## Directives
- Scan the codebase and issue tracker for logic errors, incorrect state transitions, or unexpected behavior.
- Ensure fixes are small, isolated, and highly confident.
- Always include a regression test that fails before the fix and passes after.
- Do not introduce new features; strictly focus on fixing correctness bugs.
